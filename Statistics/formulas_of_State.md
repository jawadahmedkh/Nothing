# 1. Mean (Arithmetic Mean)

$$
\text{Mean} = \frac{\sum x_i}{n}
$$

where:

* $\sum x_i$ = sum of all values
* $n$ = total number of values

---

# 2. Median

* **Step 1:** Arrange values in **ascending order**.
* **Step 2:**

  * If $n$ is **odd**:

    $$
    \text{Median} = \text{Middle value}
    $$
  * If $n$ is **even**:

    $$
    \text{Median} = \frac{\text{Middle two values' sum}}{2}
    $$

---

# 3. Mode

**Formula:**

* **Mode** = Most Repeated Value

**Note:** **If frequency table is given, for **Grouped Data** then:**

$$
\text{Mode} = L + \left( \frac{(f_1 - f_0)}{(2f_1 - f_0 - f_2)} \right) \times h
$$

where:

* $L$ = lower boundary of modal class
* $f_1$ = frequency of modal class
* $f_0$ = frequency of class before modal class
* $f_2$ = frequency of class after modal class
* $h$ = class width

---

# 4. Population Mean

**Formula:**

$$
  \mu = \frac{\sum x_i}{N}
  $$

  where:

  * $\sum x_i$ = sum of all values in the population
  * $N$ = total number of individuals in the population

---

# 5. Sample Mean

**Formula:**

$$
\bar{x} = \frac{\sum x_i}{n}
$$

where:

* $\sum x_i$ = sum of all values in the sample
* $n$ = number of values in the sample

---

# 6. Range

**Formula:**

$$
\text{Range} = \text{Maximum value} - \text{Minimum value}
$$

# 7. Population Vairance

**Formula:**

$$
\sigma^2 = \frac{\sum (x_i - \mu)^2}{N}
$$

where:

* $x_i$ = individual data points
* $\mu$ = population mean
* $N$ = total number of data points

---

# 8. Sample Variance

**Formula:**

$$
s^2 = \frac{\sum (x_i - \bar{x})^2}{n - 1}
$$

where:

* $x_i$ = individual data points
* $\bar{x}$ = sample mean
* $n$ = number of data points in the sample

---

# 9. Population Standard Deviation
* **Formula:**

$$
\sigma = \sqrt{\frac{\sum (x_i - \mu)^2}{N}}
$$

# 10. Sample Standard Deviation
* **Formula:**

  $$
  s = \sqrt{\frac{\sum (x_i - \bar{x})^2}{n - 1}}
  $$

---

# 11. IQR (Inter Quartile Range)

**Formula:**

$$
\text{IQR} = Q3 - Q1
$$
